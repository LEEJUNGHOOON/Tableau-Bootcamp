
**📌 Why Tableau?**

•	**사명**: 사람들이 데이터를 **보고 이해**할 수 있도록 돕는 것이 목적

•	**분석 철학**: 단발성 시각화로 끝내는 것이 아닌, **지속적 분석과 인사이트 도출**

•	**이점**:

•	**개인**: 시간 절약

•	**팀**: 신속한 의사결정

•	**조직**: 데이터 기반 기업 문화 정착

---

**🛠️ Tableau 제품군 (플랫폼 구성)**

| **제품명** | **역할** |
| --- | --- |
| Tableau Prep Builder | 데이터 정리, 결합, 변형 |
| Tableau Desktop | 데이터 분석 및 시각화 |
| Tableau Server / Online | 결과 공유 및 협업 |
